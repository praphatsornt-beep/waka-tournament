## 1. ลูกค้า (Customer)

### 1.1 สั่งซื้อสินค้า
```
เปิดลิงก์สั่งซื้อใน LINE → เลือกสินค้า + จำนวน → เลือกสาขา
→ กรอกชื่อ + เบอร์โทร → แนบสลิป → ส่งออเดอร์
→ ได้ LINE: "รับออเดอร์แล้ว #XXXXXX"
```

**สำคัญ:** ลูกค้าต้องเปิดลิงก์ **ใน LINE** เท่านั้น ไม่ใช่ browser — ถ้าเปิดใน browser จะไม่ได้รับ LINE notification

- จำกัดจำนวนสั่งซื้อตาม limit ใน `_catalog` คอลัมน์ J/K

### 1.2 รับสินค้า
```
ได้ LINE: "สินค้าพร้อมรับที่สาขา" → ไปสาขา
→ พนักงานกดส่งมอบ → ได้ LINE: "กรุณากดยืนยันรับของ"
→ กดลิงก์ยืนยัน → จบ ✅
```

## 2. คลังกลาง — กัน + จั้ม + น้องใหม่

**เข้าที่:** https://waka-liff.vercel.app/app.html → ใส่ `waka99`

### 2.1 รับสินค้าเข้าคลัง (กัน / จั้ม)

**แท็บ: 📦 รับเข้า**

```
สินค้ามาถึงคลัง → แท็บ รับเข้า → สแกน barcode หรือเลือกสินค้า
→ ใส่จำนวน Box/Pack ที่เพิ่ม → ตั้ง limit ขายออนไลน์ (ถ้าต้องการ)
→ กด "เพิ่มสต็อก"
```

**เพิ่มสินค้าใหม่:**
```
แท็บ รับเข้า → เพิ่มสินค้าใหม่ → สแกน barcode (ไม่บังคับ)
→ กรอกชื่อ, หมวดหมู่ (dropdown), ราคาขาย, ราคาทุน
→ จำนวนเริ่มต้น, จำกัดขายออนไลน์ → กด "เพิ่มสินค้าใหม่"
→ เพิ่มใน _catalog + stock อัตโนมัติ
```

### 2.2 สร้างล็อตส่งสาขา (กัน / จั้ม)

**แท็บ: 🏭 คลัง → สร้างล็อต**

```
เลือกสาขา → เห็นรายการสินค้า + จำนวนออเดอร์อัตโนมัติ
→ กรอกจำนวนเผื่อ → เห็น "รวมส่ง" อัปเดตทันที
→ กด "สร้างล็อต" → ตัดสต็อกกลางอัตโนมัติ
→ LINE แจ้งกลุ่ม staff + ลิงก์ให้สาขากดรับของ
```

### 2.3 นับของ & จัดส่ง (น้องใหม่)

**แท็บ: 🏭 คลัง → ประวัติ**

```
ดูรายการล็อตที่ต้องส่ง (สถานะ: จัดส่ง)
→ นับของตามรายการ → จัดส่งไปสาขา
```

### 2.4 ตรวจสลิปที่มีปัญหา

**แท็บ: 🏠 Home**

```
เห็นออเดอร์ล่าสุด → กดออเดอร์ที่สลิปมีปัญหา (สลิปซ้ำ/ยอดไม่ตรง/ฯลฯ)
→ ตรวจสอบในแอปธนาคาร
→ กด "ยืนยันสลิป + แจ้งลูกค้า" → ลูกค้าได้ LINE: "ยืนยันการชำระเงินแล้ว ✅"
```

**สถานะสลิปที่ต้องตรวจ:**
| สถานะ | ความหมาย | สิ่งที่ต้องทำ |
|-------|----------|-------------|
| ยืนยัน | ผ่านอัตโนมัติ | ไม่ต้องทำอะไร ✅ |
| รอตรวจ | อ่านสลิปไม่ได้ | เช็คแอปธนาคาร → กดยืนยันใน Home |
| รอตรวจเพิ่ม | อ่านได้แต่ไม่ 100% | เช็คแอปธนาคาร → กดยืนยัน |
| ยอดไม่ตรง | สลิป < ยอดออเดอร์ | ติดต่อลูกค้า หรือกดยืนยัน |
| สลิปซ้ำ | เลขอ้างอิงซ้ำ | ติดต่อลูกค้า |
| บัญชีไม่ตรง | โอนผิดบัญชี | ตรวจสอบแอปธนาคาร |
| สงสัยปลอม | ตัดต่อ | ตรวจสอบด้วยตา |
| ไม่มีสลิป | ไม่ได้แนบ | ติดต่อลูกค้า |

### 2.5 ดูแท็บอื่นๆ ในคลัง

| แท็บ | ใช้ทำอะไร |
|------|-----------|
| สรุปสาขา | ดูจำนวนสินค้าที่ต้องส่งแต่ละสาขา |
| สร้างล็อต | สร้างล็อตส่งสาขา |
| ประวัติ | ดูล็อตที่เคยส่ง + สถานะ (จัดส่ง/รับแล้ว) |
| สต็อก | ดูสต็อกกลาง + สต็อกแต่ละสาขา |

## 3. พนักงานสาขา

**เข้าที่:** https://waka-liff.vercel.app/app.html → ใส่รหัสสาขา (`ts01` / `mt01` / `sn01`)

### 3.1 รับของจากคลัง

**เมนู: 📥 รับของจากคลัง**

```
เห็นล็อตที่คลังส่งมา + รายการสินค้า
→ เช็คว่าของครบ → กด "ยืนยันรับของครบแล้ว"
→ สต็อกสาขาเพิ่มอัตโนมัติ
→ LINE แจ้งลูกค้าทุกคนที่มีออเดอร์ที่สาขานี้: "สินค้าพร้อมรับ"
```

### 3.2 ดูออเดอร์

**เมนู: 📋 ออเดอร์**

```
เห็นรายการออเดอร์ทั้งหมดของสาขา (เฉพาะที่สลิปยืนยันแล้ว)
→ กดที่ออเดอร์ → เห็นรายละเอียด (ชื่อ, เบอร์, สินค้า, สถานะ)
→ กด "ส่งมอบสินค้า" ได้เลย
```

### 3.3 ส่งมอบลูกค้า

**เมนู: 🤝 ส่งมอบ**

```
ลูกค้ามาหน้าร้าน → ค้นหาชื่อ / เบอร์ / Order ID
→ กด "ส่งมอบ" → ตัดสต็อกสาขาอัตโนมัติ
→ LINE แจ้งลูกค้า: "กรุณากดยืนยันรับของ"
→ ลูกค้ากดยืนยัน → สถานะเป็น "รับแล้ว" ✅
```

### 3.4 ดูสต็อกสาขา

**เมนู: 📦 สต็อกสาขา**

```
เห็นสินค้าคงเหลือของสาขา (Box / Pack)
```

### 3.5 เบิกสินค้า

**เมนู: 📤 เบิกสินค้า**

```
เลือกสินค้า → ประเภท (Box/Pack) → จำนวน
→ เลือกเหตุผล (จัดกิจกรรม / ของชำรุด / ตัวอย่าง / อื่นๆ)
→ กด "เบิกสินค้า" → ตัดสต็อกสาขา + บันทึก log
```

## 5. ตั้งค่าสินค้า (_catalog)

| คอลัมน์ | ใช้ทำอะไร |
|---------|-----------|
| A: name | ชื่อสินค้า |
| B: category | หมวดหมู่ |
| C: price_box | ราคาขาย Box |
| D: price_pack | ราคาขาย Pack (ว่าง = ไม่มี) |
| E: active | TRUE/FALSE เปิด-ปิด |
| F: image_url | ลิงก์รูป Google Drive |
| G: cost_box | ต้นทุน Box |
| H: cost_pack | ต้นทุน Pack |
| I: slug | URL ย่อ เช่น SL02 |
| J: limit_box | จำกัดขาย Box (ว่าง = ไม่จำกัด) |
| K: limit_pack | จำกัดขาย Pack (ว่าง = ไม่จำกัด) |
| L: barcode | Barcode EAN/UPC |

**หมายเหตุ:** หลังแก้ข้อมูล → รัน `clearCache()` ใน GAS editor (หรือรอ 5 นาที)
